# Звіт по проєкту "Форматування Excel файлів"

## Опис файлів та директорій проєкту

У даному проєкті реалізована програма на Java з використанням бібліотеки **Apache POI** для автоматичного форматування комірок Excel залежно від їхнього вмісту. Нижче наведено опис основних компонентів проєкту:

1. **`src/main/java/com/example/ExcelFormatter.java`**
   - Головний файл із вихідним кодом програми.
   - Відповідає за:
     - Читання вихідних Excel-файлів із папки `exel_files`.
     - Аналіз тексту кожної комірки за допомогою регулярних виразів (Regex).
     - Застосування правил зафарбовування фону комірок (зелений — тільки цифри, синій — тільки літери, жовтий — букви і цифри, помаранчевий — ні того, ні іншого).
     - Збереження змінених файлів у папку `exel_changed`.

2. **`pom.xml`**
   - Конфігураційний файл збирача **Maven**.
   - Відповідає за управління структурою проєкту та автоматичне завантаження зовнішніх бібліотек (залежностей). У ньому прописані необхідні для роботи програми бібліотеки `poi`, `poi-ooxml` та `log4j`.

3. **`exel_files/`**
   - Вхідна директорія (створюється/використовується програмою).
   - Містить оригінальні неформатовані файли Excel (`.xlsx` або `.xls`), які програма буде аналізувати.

4. **`exel_changed/`**
   - Вихідна директорія.
   - Сюди програма автоматично зберігає нові файли із вже застосованим форматуванням комірок. У їхніх назвах додається суфікс `_changed`.

5. **`apache-maven-3.9.6/`** та **`maven.zip`**
   - Портативна версія збирача Maven.
   - Використовувалася для локального завантаження залежностей і запуску програми через командний рядок без необхідності глобального встановлення Maven на вашому комп'ютері.

6. **`target/`**
   - Службова папка Maven, яка створюється автоматично при запуску або компіляції. Містить скомпільовані класи Java (`.class` файли).

## Як запустити програму

Ви маєте два основних способи запуску даної програми:

### Спосіб 1: З використанням середовища розробки (IntelliJ IDEA, Eclipse тощо) — Рекомендований
1. Відкрийте вашу IDE (наприклад, IntelliJ IDEA).
2. Виберіть **Open** (Відкрити) і вкажіть шлях до папки `1_1` (або виберіть файл `pom.xml` і натисніть "Open as Project").
3. Зачекайте пару хвилин, поки IDE автоматично завантажить усі залежності, вказані в `pom.xml`.
4. У дереві проєкту (зліва) перейдіть за шляхом: `src -> main -> java -> com -> example -> ExcelFormatter`.
5. Натисніть правою кнопкою миші по файлу `ExcelFormatter` (або на зелену стрілочку біля `public static void main`) і виберіть **Run 'ExcelFormatter.main()'**.

### Спосіб 2: Через командний рядок з використанням портативного Maven (Уже налаштовано в папці)
1. Відкрийте **PowerShell** або звичайний командний рядок (`cmd.exe`).
2. Перейдіть до папки з проєктом `1_1`.
3. Скопіюйте і виконайте наступну команду (команда `chcp 65001` виправить проблеми з кодуванням та відображенням "кракозябр" кирилицею):
```cmd
cmd.exe /c "chcp 65001 > nul && .\apache-maven-3.9.6\bin\mvn.cmd -B clean compile exec:java -Dexec.mainClass=com.example.ExcelFormatter"
```
4. Дочекайтеся появи повідомлення `BUILD SUCCESS` (Готово!). Результати роботи програми (відформатовані файли) з'являться у папці `exel_changed`.
